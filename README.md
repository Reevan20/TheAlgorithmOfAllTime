A repo for collecting libraries with TheAlgorithmOfAllTime.
# python
## Install
- Download the `TAOAT.py` file.
- `import taoat.py`
# rust
## Install
- Download the crate
- Add `taoat = { path = "path/to/crate/" }`
# js
## Install
- Download `TAOAT.js`
- Add `import { is_six } from "/path/to/TAOAT.js";`
